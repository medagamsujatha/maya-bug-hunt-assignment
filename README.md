# maya-bug-hunt-assignment
Bug Hunt assignment for Maya’s Next.js e-commerce application. This repository documents identified UI/UX issues, functional bugs, performance problems, and security gaps with clear reproduction steps, severity levels, root cause analysis, and suggested fixes to demonstrate product thinking, testing skills, and technical depth.

Project Description: This repository contains the Maya Bug Hunt assignment focused on testing a Next.js e-commerce application. It includes identification and documentation of UI/UX issues, functional bugs, performance bottlenecks, and security gaps, along with reproducible steps, severity classification, and suggested fixes to demonstrate product thinking and testing skills.

## Project Limitations
* The testing scope is limited to the features available in the provided Next.js e-commerce application.
* Backend source code and internal APIs are not accessible for deep server-side validation.
* Security testing is limited to basic checks; advanced penetration testing was not performed.
* Performance testing was conducted under limited network and device conditions.
* Third-party integrations such as payment gateways and authentication services were tested only at surface level.
* Testing duration was limited, so some edge cases may not have been fully explored.
* Automated testing tools were minimally used; most testing was manual and exploratory.
* Findings depend on the environment used and may vary across different devices or browsers.
